# 🧠 StudyMind AI – Intelligent Document-Based Learning Assistant

🚀 StudyMind AI is an advanced AI-powered learning assistant that allows users to interact with documents (PDFs & PPTs) using natural language. It helps students learn faster by generating answers, quizzes, flashcards, notes, and summaries automatically.

---

## ✨ Features

* 📄 Upload PDF & PPT files
* 💬 Chat with your documents (context-based answers)
* 🎯 Auto Quiz Generation (MCQs)
* 🃏 Flashcards for quick revision
* 📝 Smart Notes Generation
* 🔍 Document Summary
* ⚡ Fast responses using Groq LLM

---

## 🛠️ Tech Stack

**Frontend:**

* Streamlit

**Backend:**

* Python

**AI / ML:**

* LangChain
* Groq API (LLaMA 3)
* HuggingFace Embeddings
* Retrieval-Augmented Generation (RAG)

**Database:**

* Chroma Vector Store

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/StudyMind-AI.git
cd StudyMind-AI
```

### 2. Create virtual environment

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Set API Key

Create a `.env` file and add:

```env
GROQ_API_KEY=your_api_key_here
```

### 5. Run the application

```bash
streamlit run app.py
```
